
# 🚀 FreelanceHub-2.0

A modern full-stack freelancer marketplace platform that connects recruiters with skilled freelancers through a scalable, database-driven workflow.

Built using Node.js, Express.js, MySQL, and Vanilla JavaScript, the platform supports dynamic freelancer discovery, project management, filtering, sorting, and real-time frontend-backend-database integration.

---

# ✨ Features

- Dynamic freelancer registration system
- Recruiter project posting workflow
- Real-time freelancer browsing
- Search freelancers by skills or name
- Category-based filtering
- Location-based filtering
- Hourly rate filtering
- Availability filtering
- Dynamic sorting system
- REST API integration
- MySQL-powered dynamic rendering
- Responsive UI/UX
- Email integration using Web3Forms

# 🔄 Workflow

## Freelancer Registration

1. User submits registration form  
2. Backend validates request  
3. User and freelancer data are inserted into MySQL  
4. Freelancer cards dynamically render on the platform  

---

## Project Posting

1. Recruiter posts project  
2. Backend stores project data  
3. Projects page updates dynamically  

---

## Project Application

1. Freelancer clicks “Apply Now”  
2. Application request is processed  
3. Recruiter receives application through email integration  

---

# 📦 Setup

## Install Dependencies

```bash
npm install
````

## Start Backend Server

```bash
node server.js
```

## Database Setup

1. Open MySQL Workbench
2. Run SQL schema
3. Start backend server
